探索01

数据概览与质量

这一板块做什么

拿到数据先认清它本身：规模、字段、一行代表什么、质量如何。后面所有分析都建立在这一步之上。

  • 粒度与主键。一行一个房源,共 36,445 行。注意 id 列被 float64 精度损坏(房源 id 量级约 1.6×10¹⁸,超出安全整数范围,17,064 个 id 因此撞车),应改用 listing_url 当主键(36,445 个真唯一)。
  • 价格概览。价格区间 $93–300,均价 $132、中位 $120,分布高度集中——42.6% 的房源正好是 $120,高于 $200 的仅 8.3%(见下图)。
  • 地段与房型字段齐全(行政区、社区、经纬度、房型、容量),可支撑地理与房型维度的分析。
  • 缺失是结构性的。70.5% 的房源近一年零评论,其中约 45% 连评分、月评论都为空；bedrooms 缺 16%,beds/bathrooms 缺约 40%。缺失本身就是"不活跃"的信号,不宜无脑插补。
探索02

单变量分析

这一板块做什么

逐个字段单独看分布,掌握每个变量的基本形态,为后面的关系分析打底。

  • 地段高度集中。曼哈顿 44.9% + 布鲁克林 36.4% ≈ 八成；皇后区 14.6%,布朗克斯 3.1%,史泰登岛 1.0%。
  • 房型近乎二元。整租房源 53.3% vs 独立房间 45.1%,酒店 + 合住合计不到 2%。
  • 容量偏小。accommodates 中位 2(近一半房源就是 2 人),中位 1 间卧室。
  • 可订性两极分化。availability_365 有 33.5% 是 0(未来一年完全不可订),又有 36% 落在 181–364 天的高可订区。一边死库存,一边几乎全年挂着。
  • 活动度普遍极低。近 12 月评论中位为 0,70.5% 的房源近一年零评论；有评分的房源中位 4.86(高分通胀明显)。
探索03

双变量分析

这一板块做什么

把字段两两组合看关系——价格、地段、房型、可订性之间怎么联动。

价格随地段和房型变化。整租房源在每个区都比独立房间贵约 15–30%；分区看,曼哈顿整体均价最高($139),皇后区最低($121)。两者叠加,曼哈顿整租均价 $147,而皇后区独立房间只有 $114。

房型构成也随地段变化。曼哈顿整租占 64%(全市最高),越往外围越以独立房间为主(皇后、布朗克斯整租仅约 36%)——曼哈顿是"整租公寓"市场,外围是"分租房间"市场。

  • 活跃房源是少数。把"可订(availability_365>0)且近一年被订过(number_of_reviews_ltm>0)"定义为活跃,只有 28.3% 满足；而且活跃率在各区(27–36%)、各房型(都约 28%)之间几乎一样平。
  • 活跃房源内部出现反转。曼哈顿活跃房源的月评论中位最低(0.44),布鲁克林 / 皇后区更高(约 0.9)——曼哈顿高单价、低翻台,外围高翻台。
  • 房源最密的社区：Bed-Stuy、Midtown、Williamsburg、Harlem、Hell's Kitchen(布鲁克林与曼哈顿核心交替)。
探索04

多维交叉与分群

这一板块做什么

多个维度一起看,把房源切成可识别的群体,看清结构。

用"可订性 × 近期活动"两个维度,把全部 36,445 个房源切成四群：

  • 真活跃只占 28.3%。最大的一群(38.1%)是"可订但近一年零评论"——挂在平台上却没人订(其中混着少量新上架房源)；32.4% 是彻底的僵尸；"订满 / 暂停"仅 1.1%,可忽略。
  • 这个分布在 房型 × 行政区 的每个格子里都相近(活跃率全在 26–36%)。正常市场波动会让僵尸集中在某些差地段或差房型；它如此均匀,更像一次全市性政策冲击的结果。
洞察05

核心洞察

这一板块做什么

把前面四块的规律收敛成几条解释现象的关键结论。

1

价格结构清晰但温和。整租处处高于独立房间,曼哈顿整体最高($139);但价格高度集中在 $120 附近,高价长尾很薄(>$200 仅 8.3%)——这是一个"中端为主、溢价有限"的市场。

2

房源数严重高估了真实市场。36k 房源里只有约 1 万(28%)真正在运营,最大群体是挂着却从未成交的房源。

3

僵尸化是全市均匀的。这与 2023 年 9 月生效的纽约《本地法 18 号》(短租登记法,未登记房源基本无法合法做 30 晚以下短租)高度吻合——大量未登记房源沦为死库存,横跨所有行政区与房型。(基于数据形态 + 已知监管的推断。)

4

真正在跑的是一个更小、集中、二元的市场：曼哈顿核心 + 布鲁克林为主、整租与独立房间各半、多为 2 人小户;曼哈顿高价低翻台,外围高翻台。

行动06

行动建议

这一板块做什么

把结论落成可执行动作,每条都扣回上面具体的数字与字段。

① 分析前先理清字段

用 listing_url 当主键(避免 id 精度撞车);几列派生的辅助字段(dominant_*、external_sample_count)在房源级分析里可忽略。

② 市场 / 运营分析先过滤到活跃房源

用 active = availability_365>0 且 number_of_reviews_ltm>0 过滤到约 1.03 万房源再分析,别用 36k 全量(会被 72% 的死库存稀释)。active 是真运营的下限:桶 B 里混着新房源。

③ 价格分析用均价、按房型 × 地段拆

价格取值集中在 $120,中位会被尖峰主导,做分组比较时优先看均价;重点看整租溢价(每区 +15–30%)和分区差异(曼哈顿最高)。

④ 供给 / 监管视角用全量

看死库存占比(约 72%)及其在各区各房型的均匀性,作为政策影响的指标。

⑤ 地理投放

活市场集中在曼哈顿核心(Midtown、Hell's Kitchen、UWS、UES、East Village)和布鲁克林(Bed-Stuy、Williamsburg、Bushwick、Crown Heights);曼哈顿打整租高端低频,布鲁克林 / 皇后区打高频。

进阶07

Superhost 真的更贵吗

这一板块做什么

把价格(原表)与 host_is_superhost(新接入)交叉,看"超赞房东"标签是否对应更高标价。

  • 整体几乎没有溢价。Superhost 均价 $134.2 vs 普通房东 $131.8,只贵 1.8%,中位数都是 $120。
  • 按房型拆开才看出分化。酒店房 Superhost 贵 23%、整租贵 4.7%;但独立房间反而 便宜 3%、合住便宜 1.3%。
  • 结论:Superhost 不靠"贵",而靠"同价更好"——口碑优势没有体现在标价上。
进阶08

房东越商业化,价格和合规怎么变

这一板块做什么

用 calculated_host_listings_count 把房东分成个人 / 小型 / 中型 / 大型职业四档,看价格、License 合规率、活跃度如何随之变化。

  • 大型职业房东(>20 套,7,019 套)定价最高 $159(比个人房东高 28%),但近 12 月评论仅 1.5、活跃度全场最低——像是"囤大量高价房挂着、低周转"。
  • 合规率不是越大越高。中型(6–20 套)最高 24.7%,大型职业反而掉到 13%,个人房东最低 9.8%。
  • 真正在跑量的是小型 / 中型房东(2–20 套),近 12 月评论约 6,远高于个人和大型职业。
进阶09

地段值多少钱

这一板块做什么

用经纬度(两表都有)按 Haversine 公式现场算出到大学 / 旅游核心 / 华人社区 / 机场的综合便利性评分,再与价格做相关——这是合并经纬度与价格后第一次能算的关系。

  • 便利性与价格中等正相关(Spearman 0.445):便利性评分越高,均价大体从约 $127 抬到 $145+。
  • 但相关只有 0.44,说明价格不只看地段——同一便利度内部,价格差异依然很大。地段是定价的一个因子,而非全部。
进阶10

贵,贵在哪个评价维度

这一板块做什么

把价格与四个细分评分(总评 / 位置 / 清洁 / 性价比)分别求相关,看高价对应哪种口碑。

  • 四个维度与价格相关都很弱:位置 0.12 最高 > 总评 0.10 > 清洁 0.07 > 性价比 0.02 几乎为零。
  • 位置相关最高,呼应"地段值钱";性价比(value)与价格几乎无关——贵房源并没有换来更差的性价比口碑,标价大体被市场接受。
  • 反直觉的一笔:最低价档($93–106)的性价比评分(4.37)反而低于中价档($106–120,4.73)——最便宜并不等于体验最划算。
进阶11

综合推荐榜(规则加权)

这一板块做什么

不动用 LLM,用规则把"便利性 0.30 + 总评 0.30 + 近期活跃 0.20 + 性价比 0.20"合成一个 0–1 的推荐分,给出推荐社区与房源,作为后续推荐网站的雏形。

  • 推荐分最高的社区高度集中在曼哈顿下城 / 中城:Nolita、Chinatown、Little Italy、Theater District、Financial District、SoHo……各社区分数很接近(0.73–0.76),差距不大。
  • Top 房源几乎全是 Midtown / Theater District / Financial District / Tribeca 的独立房间——便利、活跃、评分高,且不算最贵。

洞察延伸

延伸部分的四条结论

这一板块做什么

把合并数据后新增的四块交叉分析收敛成可汇报的关键结论。

1

Superhost 靠口碑不靠溢价。整体只贵 1.8%、中位同价;唯一明显溢价的是酒店房(+23%)——超赞房东标签更多体现在"同价更好",而非更高标价。

2

大型职业房东是"高价低周转"群体。定价最高(+28%)、活跃度最低(近年评论仅 1.5),且 License 合规率(13%)还不如中型房东(24.7%)。这正是"运营与监管"故事线的现成切入点。

3

地段值钱,但不是全部。便利性与价格 Spearman 0.445;评分维度里位置与价格相关最高(0.12),而性价比几乎为零(0.02)——高价主要买的是位置,且没有换来更差的性价比口碑。

4

推荐价值集中在曼哈顿下城 / 中城。规则加权推荐榜里 Nolita、Chinatown、Theater District、FiDi 等社区领先,但彼此分数接近,真正区分还需叠加个人偏好与预算。
